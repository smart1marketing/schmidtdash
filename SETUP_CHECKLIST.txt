### Create Render Account
☐ Go to render.com
☐ Sign up (can use GitHub login)
☐ Verify email

### Create Web Service
☐ Click "New +" button
☐ Select "Web Service"
☐ Connect GitHub account (if prompted)
☐ Select repository: ecwid-dashboard
☐ Make sure it's PUBLIC on GitHub!

### Configure Service
☐ Name: ecwid-dashboard
☐ Region: (default is fine)
☐ Runtime: Node
☐ Build Command: npm install
☐ Start Command: npm start
☐ Instance Type: Free (sufficient)
☐ DO NOT click "Create Web Service" yet!

### Add Environment Variables
☐ Click "Environment" in left sidebar
☐ Add first variable:
  ☐ Key: ECWID_STORE_ID
  ☐ Value: [paste your Store ID from Step 1]
☐ Add second variable:
  ☐ Key: ECWID_API_TOKEN
  ☐ Value: [paste your API Token from Step 1]
☐ Add third variable:
  ☐ Key: NODE_ENV
  ☐ Value: production
☐ Click "Save Changes"

### Deploy Service

## PHASE 4: HIGHLEVEL EMBEDDING (5 minutes)

### Add HTML Code Block
☐ Log in to HighLevel
☐ Navigate to your website/page
☐ Add a new "HTML Code Block" element
☐ Paste the embedding code:

```html
<div id="ecwid-dashboard"></div>
<script>
  const API_URL = 'https://ecwid-dashboard-XXX.onrender.com/api/dashboard';
  
  fetch(API_URL)
    .then(r => r.json())
    .then(data => {
      let html = `
        <div style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif; max-width: 800px;">
          <h2>Ecommerce Dashboard</h2>
          
          <div style="display: grid; grid-template-columns: 1fr 1fr; gap: 15px; margin-bottom: 20px;">
            <div style="background: #f0f4ff; padding: 15px; border-radius: 6px;">
              <p style="margin: 0; color: #666; font-size: 12px; text-transform: uppercase;">Orders This Week</p>
              <p style="margin: 5px 0 0; font-size: 24px; font-weight: bold;">${data.weekData.ordersThisWeek}</p>
              <p style="margin: 5px 0 0; font-size: 13px; color: ${data.weekData.orderChangePercent >= 0 ? '#10b981' : '#ef4444'};">
                ${data.weekData.orderChangePercent >= 0 ? '↑' : '↓'} ${Math.abs(data.weekData.orderChangePercent)}% vs last week
              </p>
            </div>
            
            <div style="background: #fff4e6; padding: 15px; border-radius: 6px;">
              <p style="margin: 0; color: #666; font-size: 12px; text-transform: uppercase;">Orders This Month</p>
              <p style="margin: 5px 0 0; font-size: 24px; font-weight: bold;">${data.monthData.ordersThisMonth}</p>
              <p style="margin: 5px 0 0; font-size: 13px; color: ${data.monthData.orderChangePercent >= 0 ? '#10b981' : '#ef4444'};">
                ${data.monthData.orderChangePercent >= 0 ? '↑' : '↓'} ${Math.abs(data.monthData.orderChangePercent)}% vs last month
              </p>
            </div>
          </div>

          <h3>Top 10 Products This Week</h3>
          <div style="display: grid; gap: 8px;">
            ${data.topProductsThisWeek.map((p, i) => `
              <div style="background: white; padding: 12px; border-left: 3px solid #3b82f6; border-radius: 4px;">
                <strong>${i + 1}. ${p.name}</strong> - $${p.price}
              </div>
            `).join('')}
          </div>

          <p style="color: #999; font-size: 12px; margin-top: 20px;">
            Last updated: ${new Date(data.lastUpdated).toLocaleString()}
          </p>
        </div>
      `;
      document.getElementById('ecwid-dashboard').innerHTML = html;
    })
    .catch(e => {
      document.getElementById('ecwid-dashboard').innerHTML = '<p style="color: red;">Error loading dashboard</p>';
      console.error(e);
    });
</script>
```

☐ Replace URL with your actual Render URL from Phase 3
☐ Save element
☐ Preview page
☐ Data should display correctly
☐ Check in mobile preview too
☐ Publish page

## TROUBLESHOOTING QUICK FIXES

### Widget shows nothing
☐ Check browser console (F12) for errors
☐ Verify Render URL is correct
☐ Test /api/dashboard endpoint directly
☐ Check Render logs for errors
☐ Verify ECWID_STORE_ID and ECWID_API_TOKEN are set

### Data is old or missing
☐ Wait for 2 AM UTC for automatic refresh
☐ Manual refresh: curl -X POST https://[YOUR_URL]/api/fetch-now
☐ Check Render logs for fetch errors
☐ Verify Ecwid API token is still valid
☐ Check if Ecwid has new API requirements

### Service keeps restarting
☐ Check Render logs for error messages
☐ Check server.js syntax
☐ Verify all dependencies in package.json
☐ Check Node.js version matches

### 404 errors
☐ Verify file paths are correct
☐ Check .gitignore isn't excluding needed files
☐ Rebuild service if needed
