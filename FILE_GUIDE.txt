## CUSTOMIZATION

### Change the 2 AM refresh time
Edit server.js, find this line:
```
cron.schedule('0 2 * * *', () => {
```
Change '0 2' to your desired time (UTC)
Examples:
- '0 5 * * *' = 5 AM UTC
- '30 3 * * *' = 3:30 AM UTC
- '0 0 * * *' = Midnight UTC

### Customize widget styling
Edit embed-widget.html
- Colors: search for #3b82f6 (blue), #8b5cf6 (purple)
- Fonts: modify font-family declarations
- Layout: adjust grid-template-columns
- Spacing: modify padding and gap values

### Add more metrics
Edit server.js
- Add new API calls to fetchAllData()
- Store values in the data object
- Return in /api/dashboard response
- Display in widget HTML
